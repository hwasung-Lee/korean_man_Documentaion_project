# 이름
pwd - 현재/작업 디렉토리명을 출력합니다.
# 사용법
pwd [옵션]...
# 설명
현재 작업 디렉토리의 전체 이름을 출력합니다.
## -L, --logical
심볼릭 링크를 포함하였을 경우라도 환경 변수에서 PWD를 사용합니다.
## -P, --physical
모든 심볼릭 링크를 피합니다.
## --help
도움말을 출력하고 종료합니다.
## --version
버전 정보를 출력하고 종료합니다.
